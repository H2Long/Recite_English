## 1. 什么是背单词软件

这是一款基于 Raylib 的桌面端背单词工具，纯绿色免安装，解压即用。它提供了完整的学习链路：

```
学单词 → 背单词（闪卡） → 测试 → 统计进度
```

所有数据保存在本地文本文件中，你可以随时用记事本编辑词库。

## 2. 如何获取软件

### 方式一：下载编译好的压缩包（推荐新手）

打开项目 Releases 页面：

> **https://github.com/H2Long/Recite_English/releases**

找到最新版本，下载对应平台的压缩包：

| 平台 | 下载文件 | 说明 |
|------|---------|------|
| **Windows** | `Recite_English-v5.0.0-win64.zip` | 解压后双击 `main_c.exe` |
| **Linux** | `Recite_English-v5.0.0-linux.tar.gz` | 解压后终端执行 `./main_c` |

> ⚠️ 如果 Releases 页面是空的，说明还没有发布编译好的版本。
> 你可以用下面的"方式二"自己编译，或联系开发者发布。

### 方式二：自己编译（适合有编程基础的用户）

需要先安装编译工具，详见 [docs/PACKAGE.md](PACKAGE.md)。

```bash
# 1. 克隆代码
git clone git@github.com:H2Long/Recite_English.git
cd Recite_English

# 2. 编译（Linux）
./build.sh package

# 3. 运行
cd build/dist
./main_c      # Linux
# 或双击 main_c.exe（Windows）
```

### 方式三：直接在 GitHub 上浏览

如果你只是想看看代码长什么样，直接访问：

> https://github.com/H2Long/Recite_English

## 3. 如何运行

### Linux 用户

```bash
# 解压
tar xzf Recite_English-v5.0.0-linux.tar.gz
cd dist

# 运行
./main_c
```

### Windows 用户

1. 解压压缩包
2. 进入 `dist` 文件夹
3. **双击 `main_c.exe`**

> 如果提示 "VCRUNTIME140.dll 丢失"：
> 下载安装 https://aka.ms/vs/17/release/vc_redist.x64.exe

### 首次启动

打开软件后，你会看到一个**登录页面**：

1. **已有账号** → 直接输入用户名密码登录
2. **没有账号** → 点击"注册账号"创建一个
3. 默认有一个管理员账号：`admin / admin`

登录后才能使用所有功能，学习进度会按账号独立保存。

## 4. 开始学习

登录后你会看到**主菜单**页面，包含欢迎信息和三个功能入口。

### 推荐学习流程

```
第1步：学单词          → 浏览全部单词，查看释义和例句
第2步：卡片背单词      → 闪卡翻转记忆，认识3次标记已掌握
第3步：选词背单词      → 看释义选单词，检验学习成果
第4步：测试模式        → 看单词选释义，查看正确率
第5步：学习进度        → 查看统计数据和每个单词的掌握情况
```

### 侧边栏导航

左侧的侧边栏是导航菜单：

```
主菜单
├── 学单词
├── 背单词
│   ├── 卡片背单词
│   ├── 选词背单词
│   └── 测试模式
├── 查找单词
├── 学习计划
│   ├── 计划管理
│   └── 学习进度
├── 设置
├── 账号管理
└── 词库管理
```

- 点击菜单项进入对应页面
- 非首页时，侧边栏顶部有 **"返回"** 按钮
- 右上角显示当前登录用户，点击可进入账号管理

## 5. 功能详解

### 📖 学单词

左侧是单词列表（可滚动），右侧显示选中单词的详细信息：

- 单词 / 音标 / 释义 / 例句
- 学习统计（认识次数、不认识次数、掌握状态、上次复习时间）
- 勾选"只显示未掌握"可以过滤已掌握的单词
- 点击"上一个"/"下一个"切换单词

左侧列表中的彩色竖条表示熟练度：
- 🟢 绿色 → 已掌握（认识 ≥ 3 次）
- 🟠 橙色 → 学习中
- 🔴 红色 → 待学习

### 🃏 卡片背单词

- 正面显示英文单词，**点击卡片翻转**查看释义
- 翻转后，卡片下方出现"认识"和"不认识"按钮
- 点击 **"认识"** → 认识次数 +1，认识 3 次后标记为"已掌握"
- 点击 **"不认识"** → 不认识次数 +1，重置掌握状态
- 卡片区域上方有进度条，显示当前进度
- 下方显示本轮统计（认识 / 不认识数量）

### ✍️ 选词背单词

- 显示汉语释义作为题干
- 从 4 个英文选项中选出正确的单词
- 答对绿色高亮，答错红色高亮
- 每轮最多 15 题
- 完成后显示正确率，得分会累计到当前登录账号
- 评价体系：
  - ≥ 90% → "太棒了！继续保持！"
  - ≥ 70% → "不错！再接再厉！"
  - ≥ 50% → "需要多复习哦！"
  - < 50% → "建议先去学单词模式"

### 📝 测试模式

- 显示英文单词，从 4 个释义选项中选出正确答案
- 每轮最多 10 题
- 完成后显示正确率和评价

### 🔍 查找单词

- 支持通配符模式搜索
- 语法：
  - `^abc` → 以 abc 开头的单词
  - `abc$` → 以 abc 结尾的单词
  - `a*c` → 包含 a 和 c，中间任意字符
  - `ab?d` → a b + 任意 1 字符 + d
  - 直接输入文本 → 不区分大小写的子串匹配
- 搜索结果会显示每个单词的完整信息（单词、音标、释义、例句）

### 📅 学习计划

**计划管理页面：**
- 左侧列表显示所有计划，点击激活
- 右侧表单可以创建新计划（名称 + 每日词数 + 总天数）
- 红色 ✕ 按钮删除计划
- "使用默认"按钮添加 4 个默认计划：

| 计划名称 | 每日词数 | 周期 |
|---------|---------|------|
| 一周入门计划 | 10 词 | 7 天 |
| 半月巩固计划 | 15 词 | 15 天 |
| 三十天进阶计划 | 20 词 | 30 天 |
| 六十天冲刺计划 | 30 词 | 60 天 |

**学习计划首页：**
- 显示当前激活计划、第几天、今日已学/目标
- 进度条直观显示完成情况

### 📊 学习进度

- 统计卡片：总单词数、已掌握、学习中
- 详细列表：每个单词的认识次数、不认识次数、掌握状态、上次复习时间
- 可按"清除进度"按钮重置所有学习数据

### 🌓 设置

- **深色模式**：勾选复选框切换深色/浅色主题
- 当前版本信息

### 👤 账号管理

- **已登录**：显示用户信息、注册时间、可登出或切换账号
- **未登录**：显示登录/注册入口
- 下方显示所有已注册用户列表
- 每个用户的学习进度和计划独立保存

### 📋 词库管理

- 左侧搜索框搜索单词（自动匹配）
- 点击单词选中，右侧显示编辑表单
- 可编辑单词的：单词、音标、释义、例句、例句翻译
- 支持添加新单词和删除单词
- 修改后自动保存到 `data/words.txt`

## 6. 常见问题

### Q: 如何添加自己的单词？

两种方法：
1. **软件内** → 进入"词库管理"页面，点击"新增"按钮
2. **直接编辑文件** → 用记事本打开 `data/words.txt`，按格式添加：

```
单词|音标|词性.释义|例句|例句翻译
```

示例：
```
advance|[əd'væns]|v. 前进；进步|The army began to advance.|军队开始前进。
```

### Q: 学习进度存在哪里？

每个用户的学习进度独立保存：

```
data/progress.txt           ← 默认（未登录）
data/progress_用户名.txt    ← 登录后保存为用户名专属文件
```

### Q: 如何备份/迁移学习数据？

直接复制整个 `data/` 文件夹即可。里面包含词库、账号、进度和计划。

### Q: 软件支持哪些操作？

| 操作 | 说明 |
|------|------|
| 鼠标点击 | 选择菜单、点击按钮、选择选项 |
| 鼠标滚轮 | 滚动列表 |
| 键盘输入 | 在输入框中输入文本（支持中文） |
| 退格键 | 删除光标前的字符 |
| Delete 键 | 删除光标后的字符 |
| 方向键 ← → | 在输入框中移动光标 |
| Esc 键 | 退出程序 |

### Q: 软件是免费的吗？

是的，完全免费开源。代码在 GitHub 上可以查看和修改。
