# 背单词软件 (raylib-word)

> 基于 Raylib 的桌面端背单词应用，支持三种学习模式、多用户账号、学习计划管理

---

## 软件简介

**背单词软件** 是一个轻量级、全功能的桌面背单词工具，使用 C 语言和 Raylib 图形库开发。它提供了从"学单词"到"背单词"再到"测试"的完整学习链路，同时支持多用户独立进度、自定义学习计划、深色模式等功能。

**适用场景**：
- 英语学习者日常背单词
- 课间/碎片时间快速复习
- 自制词库进行专项记忆

---

## 功能特性

### 核心学习模式

| 模式 | 功能 | 说明 |
|------|------|------|
| 📖 **学单词** | 列表浏览 + 详细释义 | 左侧可滚动单词列表，右侧显示音标、释义、例句、学习统计，支持过滤已掌握的单词 |
| 🃏 **卡片背单词** | 闪卡翻转记忆 | 正面显示单词，点击翻转查看释义，点击"认识"/"不认识"记录进度，认识 3 次自动标记已掌握 |
| ✍️ **选词背单词** | 看释义选单词 | 显示汉语释义，从 4 个英文选项中选出正确单词，得分按账号累计 |
| 📝 **测试模式** | 看单词选释义 | 显示英文单词，从 4 个释义选项中选出正确答案，完成后显示正确率和评价 |

### 辅助功能

| 功能 | 说明 |
|------|------|
| 🔍 **查找单词** | 支持正则表达式和模糊搜索两种模式 |
| 📋 **词库管理** | 在线添加、编辑、删除单词，即时保存 |
| 👤 **多用户系统** | 注册/登录/切换账号，每个用户独立保存学习进度和计划 |
| 📅 **学习计划** | 创建自定义计划（每日词数 × 总天数），自动检测每日学习状态 |
| 📊 **学习进度** | 可视化统计卡片 + 每个单词的详细进度列表（认识/不认识次数、状态、上次复习时间） |
| 🌓 **深色模式** | 一键切换深色/浅色主题 |
| 📎 **词库可扩展** | 纯文本文件存储单词（`word|phonetic|definition|example|translation`），可直接编辑 |

## 快速开始

### 环境要求

- CMake ≥ 3.10
- Raylib 库
- C11 兼容编译器（GCC/Clang）
- 中文字体（自动检测系统字体，或使用项目自带的 NotoSansCJK.otf）

### 编译运行

```bash
# 安装依赖（Ubuntu/Debian）
sudo apt install libraylib-dev cmake ninja-build

# 编译
cd main.c
cmake . -G Ninja
ninja

# 运行
./main_c
```

## 技术特点

- **纯 C11 编写** — 无 C++ 依赖，编译快速，体积小巧
- **Raylib 图形库** — 轻量级游戏库，自带窗口管理、输入处理、字体渲染
- **UTF-8 全支持** — 所有文本输入和显示均基于 UTF-8，完美支持中文输入法
- **混合字体渲染** — 逐字符判断 ASCII / IPA 音标 / 中文，自动选择对应字体
- **状态统一管理** — `AppState` 集中管理所有全局状态，通过访问器函数安全访问
- **模块独立设计** — `account` 和 `plan` 模块支持内部备用状态，可独立运行也可集成
- **纯文本持久化** — 所有数据使用 `|` 分隔的纯文本文件，方便查看和手动编辑

## 操作说明

### 基本操作

| 操作 | 说明 |
|------|------|
| 鼠标点击 | 选择菜单、点击按钮、选择选项 |
| 鼠标滚轮 | 滚动列表 |
| 键盘输入 | 在输入框中输入文本（支持中文） |
| 退格键 | 删除输入框中光标前的字符 |
| Delete 键 | 删除输入框中光标后的字符 |
| 左右方向键 | 在输入框中移动光标 |

### 页面导航

```
主菜单
├── 学单词
├── 背单词
│   ├── 卡片背单词
│   ├── 选词背单词
│   └── 测试模式
├── 查找单词
├── 学习计划
│   ├── 计划管理
│   └── 学习进度
├── 设置
├── 账号管理（含登录/注册）
└── 词库管理
```

左侧侧边栏显示当前页面下的子菜单，非首页时显示"返回"按钮。

### 账号机制

- 首次运行自动创建默认账号 `admin / admin`
- 每个用户拥有独立的进度文件和计划文件（如 `progress_hhlong.txt`、`plans_hhlong.txt`）
- 登录时自动切换到用户专属数据
- 未登录时启动强制登录遮罩，只能访问账号相关页面

### 学习计划

提供 4 个默认计划，也可自定义：
- 一周入门计划：10 词/天 × 7 天
- 半月巩固计划：15 词/天 × 15 天
- 三十天进阶计划：20 词/天 × 30 天
- 六十天冲刺计划：30 词/天 × 60 天

计划会自动检测新的一天，累加天数并重置每日学习计数。
